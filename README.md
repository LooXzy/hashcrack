# HashCrack
> Script to crack 256 hashes with salts.  
> - Author : LooXzy
> - Created on 27/10/2022

## Documentation
### Display Help
```bash
./HashCrack -h
```

### Run Command
```bash
./HashCrack -W $path_wordlist -H $hash -S $salt
```
